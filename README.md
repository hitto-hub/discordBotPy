# discordBotPy

py-cordをつかったdiscord botのサンプルです。

## envファイル

```env
TOKEN="discord botのトークン"
APPID="天気apiのappid"
```

## pip

```bash
pip3 install -r requirements.txt
```

## 実行

```bash
python3 main.py
```
